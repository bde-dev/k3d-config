# Prometheus Lab

`k3d` spec to spin up a cluster for exploring Prometheus fundamentals — labels, cardinality, scrape config, recording rules, and alerting rules — via the `kube-prometheus-stack` chart.

`prometheus-lab-install.yaml` has been generated by running the following:

```bash
helm template prometheus-lab prometheus-community/kube-prometheus-stack --namespace monitoring --create-namespace --include-crds --values values.yaml > prometheus-lab-install.yaml
```

Frontend accessible on [http://localhost:4006](http://localhost:4006)

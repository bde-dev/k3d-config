# OpenTelemetry Operator

`k3d` spec to spin up a cluster to demonstrate the OpenTelemetry operator.

`otel-operator-install.yaml` has been generated by running the following:

```bash
helm template otel-operator open-telemetry/opentelemetry-operator --namespace opentelemetry-operator-system --create-namespace --values values.yaml > otel-operator-install.yaml
```
